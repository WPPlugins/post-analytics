== Description ==
Post-Analytics
[Post-Analytics](http://www.dossihost.net/post-analytics/) will give you data for each page/post you are editing from Google analytics.

Post-Analytics will fetch your last 30 days data from your Google Analytics account.


Features include in the every post and pages:

Site Globals Analytics:

* Unique pageviews.
* Total Entrances.
* Avg Time On Page.
* Exit Rate.
* Avg Exit Rate.
* Avg TimeOnSite.
* Page Views Per Visit.
* Page Value.
* Site Avg Bounce Rate.

Post and Page Specific Analytics:

* Page Views.
* Unique Page Views.
* Avg Time On Page.
* Entrance Rate.
* Bounce Rate.
* Exit Rate.
* Page Value.

Search Results Analytics for the current page edited:

* Total Organic Searches traffic that used to get into this page.
* keyword used to get into this page.
* Total visitis that used to get into this page.

Notes:

* This Plugin Needs php 5.3+ 
* if you want more data pls ask we will add.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Post-Analytics'
3. Activate Post-Analytics from your Plugins page. 
4. Visit 'Setting > Post-Analytics'.

= From WordPress.org =

1. Download Post-Analytics.
2. Upload the 'Post-Analytics' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Post-Analytics from your Plugins page. 
4. Visit 'Setting > Post-Analytics'.
5. Enter your valid google account details
